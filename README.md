🌌 HELLSTAR

Landing page moderna inspirada en una estética espacial y futurista, desarrollada con HTML, CSS, JavaScript y Tailwind CSS.

✨ Características
Diseño responsive
Fondo espacial animado
Nebulosas dinámicas
Estrellas generadas por JavaScript
Estrellas fugaces automáticas
Efecto parallax
Navbar con Glassmorphism
Animaciones de scroll
Cards de productos interactivas
Despliegue con Docker y Nginx
🛠️ Tecnologías
HTML5
CSS3
JavaScript
Tailwind CSS
Docker
Nginx
Git & GitHub

🚀 Ejecutar con Docker
Construir la imagen
docker build -t hellstar .
Ejecutar
docker run -d -p 8080:80 hellstar

Abrir en:

http://localhost:8080
🚀 Ejecutar con Docker Compose
docker compose up -d --build

Detener:

docker compose down
📥 Clonar repositorio
git clone https://github.com/JUNI00R89/Hellstars-ci.git
